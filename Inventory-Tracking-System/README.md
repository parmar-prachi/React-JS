📌 Inventory Tracking System : 

The Inventory Tracking System is a web application built using React.js, Redux Toolkit, and Firebase Realtime Database. It allows businesses to manage product inventory efficiently with real-time data synchronization.

The system supports full CRUD operations—adding, viewing, updating, and deleting products—while using Redux Thunk to handle asynchronous Firebase operations with proper loading and error states. Any changes made in the database are instantly reflected in the UI without requiring a page refresh.

✅ Key Features :

Add, view, update, and delete inventory items
Real-time data synchronization with Firebase
Centralized state management using Redux Toolkit
Low stock alert support (extension feature)

🛠 Tech Stack :

Frontend: React.js
State Management: Redux Toolkit
Backend: Firebase Realtime Database
Middleware: Redux Thunk
Styling: Tailwind CSS / Material UI

📚 Use Case :

Ideal for retail stores or warehouses to track inventory, manage stock levels, and ensure real-time updates across multiple users.
