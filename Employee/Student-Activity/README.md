💻 Student Activity Manager  :: 

📝 Project Overview :- 
Student Activity Manager is a React-based interactive application that helps students practice important React concepts like events, conditional rendering, refs, fragments, and passing arguments in event handlers.

The application allows users to add students, toggle their details, mark attendance, and delete records. All operations are handled using React state and component-based UI updates, without any backend or database.

🧩 Project Structure :: 

🧱 Component Modules :-

The project is divided into reusable React components:

📝 StudentForm → Add new students using Refs + Form Events

📋 StudentList → Display students using Fragments & list rendering

🎴 StudentCard → Show individual student details with conditional rendering

Each component focuses on a specific React concept and updates the UI dynamically.

🎯 React Events Handling ::-

Users interact with the application using React events:-

✍️ onChange → Typing student names

📩 onSubmit → Adding new students

🖱️ onClick →

Toggle student details ::- 

Mark attendance

Delete student

These events demonstrate how React handles event objects and UI updates efficiently.

🎛 Passing Arguments to Event Handlers ::-

The project shows how to pass custom arguments to event handlers.

Examples: -

(id) => deleteStudent(id)
markStatus(id, "present")

This allows the application to update specific student records in the React state.

🎭 Conditional Rendering :: 

The UI dynamically changes using React conditional rendering.

Examples include:-

⚠️ Showing “No Students Found” when the list is empty

👀 Toggle show/hide student details

📊 Display attendance status

❗ Show validation messages in the form

This demonstrates how React builds dynamic interfaces.

🎛 React Refs ::-

The StudentForm component uses React Refs to control input behavior.

Refs are used to:-

🎯 Auto-focus the input field

🔄 Reset and refocus after adding a student

⚠️ Focus back when validation fails

This teaches how to access DOM elements directly in React.

🧩 React Fragments ::-

The project uses React Fragments (<> </>) to avoid unnecessary wrapper elements.

Fragments are used in:-

📋 List rendering

🎴 Student card layout

📝 Form and validation messages

This keeps the DOM clean and optimized.

📊 Dashboard Features ::-

The dashboard displays real-time student activity including:-

👨‍🎓 Total Students

✅ Present Students

❌ Absent Students

📋 Student List

👀 Toggle Details View

⚠️ Validation Messages 

All updates happen instantly through React state changes.
