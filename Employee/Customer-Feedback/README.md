💼 Customer Feedback & Issue Reporting System  ::-
📌 Project Overview :-

The Customer Feedback & Issue Reporting System is a React-based front-end application that allows users to submit structured feedback, bug reports, complaints, and suggestions.

The system includes a validated form and a dashboard where submitted feedback is instantly displayed in a card-based layout, similar to modern support ticket systems used in SaaS platforms and CRM portals.

🚀 Key Concepts Implemented ::

✔ 📝 Form Handling in React
✔ 🎛 Controlled & Uncontrolled Components
✔ ⚡ Event Handling & State Management
✔ 🔄 Dynamic Form Fields
✔ ✅ Form Validation
✔ 🎨 Form Styling
✔ 📎 Refs Usage
✔ 🔍 Conditional Rendering
✔ 🧩 Component-Based Architecture

📝 Feedback Submission Form ::

The form collects important user information including:-

👤 Full Name
📧 Email Address
📂 Issue Category (Bug, Suggestion, Complaint, Other)
🚨 Priority Level (Low, Medium, High)
📄 Detailed Description
🖼 Screenshot URL (Optional)
🗒 Additional Notes

Users can also dynamically add or remove:-

🔹 🧩 Steps to Reproduce
🔹 💡 Suggested Improvements

This simulates real issue reporting platforms like Jira, Zendesk, and GitHub Issues.

✅ Form Validation ::

The form includes professional validation such as:-

✔ Required field validation
✔ Email format validation
✔ Minimum description length
✔ Category and priority selection validation
✔ Real-time error messages
✔ Error highlighting with red borders

⚡ Event Handling

The application uses several React events:-

🔹 onChange → Update form fields
🔹 onSubmit → Handle form submission
🔹 onClick → Add or remove dynamic fields
🔹 onBlur → Trigger validation
🔹 onFocus → Apply input styling

📊 Feedback Dashboard ::

Submitted feedback appears in a dashboard with feedback cards displaying:-

👤 User Information
🏷 Category & Priority badges
📄 Description
🖼 Screenshot preview
🧩 Steps to reproduce
💡 Suggested improvements
⏱ Submission timestamp

🗂 Project Structure ::

📁 App.jsx – Central state management & data flow

📄 FeedbackForm.jsx – Handles form input and submission

🔁 DynamicList.jsx – Reusable component for dynamic fields

📋 FeedbackList.jsx – Displays all submitted feedback

🧾 FeedbackCard.jsx – Individual feedback display card

🎯 Learning Outcomes ::

This project demonstrates how real-world applications:-

✔ Build complex forms in React
✔ Validate user input properly
✔ Combine controlled & uncontrolled components
✔ Implement dynamic form fields
✔ Manage state and events efficiently
✔ Display user-generated content dynamically
